01 — Architecture Overview

Taking the stress out of budgeting.

Our platform is cleanly separated into a performant React frontend and a robust FastAPI backend[cite: 10]. It doesn't just track your spending—it actively categorizes it, analyzes your budget goals, and provides actionable, step-by-step financial advice tailored to your unique data[cite: 3].

02 — Platform Features

Intelligent capabilities.

Agentic AI Pipeline

Powered by LangGraph, our multi-agent architecture (Data Fetcher → Categorizer → Analyst → Advisor) processes your financial data to generate highly personalized financial plans[cite: 5].

Interactive Financial Charts

Visual insights using Recharts, including spending pie charts and trend area charts, instantly reflecting your financial health[cite: 6].

Secure by Default

Built on Supabase with robust Row-Level Security (RLS) policies ensuring your financial data is strictly isolated and secure[cite: 7].

Conversational AI Advisor

An interactive chat interface that lets you ask specific questions about your spending, supported by state-of-the-art LLMs via OpenRouter[cite: 8].

03 — AI Orchestration

The LangGraph Agents.

DataFetchAgent

Securely retrieves user transactions, predefined budget goals, and profile data from Supabase[cite: 17].

CategorizerAgent

Uses structured LLM tool-calling (via OpenRouter) to accurately categorize uncategorized spending[cite: 18].

BudgetAnalystAgent

Cross-references actual spending against budget goals to produce a comprehensive financial summary[cite: 19].

AdvisorAgent

Synthesizes all data to generate a tailored, actionable, and step-by-step financial advisory plan[cite: 20].

04 — System Infrastructure

Transparent structuring.

Frontend

High-performance user interface.

  • React 19 & Vite: Ultra-fast build and rendering[cite: 21].
  • Framer Motion: Fluid, physics-based UI animations[cite: 21].
  • Recharts: Beautiful, responsive data visualization[cite: 22].

Backend & AI

Robust logic and agentic workflows.

  • Python 3.11+ & FastAPI: High-performance asynchronous API framework[cite: 23].
  • LangGraph & LangChain: For building and orchestrating the multi-agent AI workflows[cite: 24].
  • OpenRouter: Access to top-tier LLMs for AI intelligence[cite: 25].

Security & Privacy

Strict data isolation protocols.

  • All financial data is heavily protected using Supabase Row-Level Security (RLS)[cite: 34].
  • Ensuring that a user can only ever read, update, or delete data linked strictly to their own authenticated user_id[cite: 34].
  • No cross-user data leakage is possible at the database level[cite: 35].
